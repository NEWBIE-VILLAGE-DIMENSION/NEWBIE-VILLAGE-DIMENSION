# 《新手村次元：一份写给冒险者的文化与视觉指南》

## 导语

**新手村次元**是立足东莞松山湖高新技术产业开发区的**全国首创游戏式运营创业孵化平台**，由连续创业者黎圣传于2025年牵头创办。平台以游戏化机制为底层设计语言，将「玩、学、就、创」四阶段成长路径嵌入青年就业创业全流程。

本文是写给每一位进入**新手村次元**的冒险者的**文化与视觉指南**，聚焦两个核心命题：价值观从何而来、如何落地为制度；视觉符号如何承载文化理念、向外部受众传递平台的精神特质与身份认同。

## 一、新手村文化

### **1.1** 平台使命

**让每一位青年都曾拥有一个属于自己的新手村。**

**新手村次元**的使命源于对**青年成长痛点**的结构性判断：大量青年群体最缺乏的并非创业意愿或潜在能力，而是一个可触及的起点——有探索兴趣但缺乏转化通道，有模糊想法但缺乏验证机制，有创业意愿但缺乏资源、经验与团队支撑。

**「新手村」**一词取自角色扮演游戏术语，指玩家首次进入虚拟世界的初始区域——门槛适中、设有基础引导任务、有成长导师、允许在低风险环境中反复试错。新手村次元将这一隐喻迁移至现实：为青年提供一个**低门槛、有指引、可试错**的成长启航原点。

### **1.2** 平台愿景

**新手村次元的发展规划**分为三个阶段：

| 阶段     | 时间        | 目标                                          |
|----------|-------------|-----------------------------------------------|
| 第一阶段 | 2025—2026年 | 以松山湖高新区为起点完成模式验证，孵化3家公司 |
| 第二阶段 | 2027—2030年 | 覆盖东莞市全域，孵化10家公司                  |
| 第三阶段 | 2031—2035年 | 扩展至10个地级市，形成区域联动的创业孵化网络  |

愿景的实现路径依托**「人才—空间—资本」三层架构**：人才层（核心创始团队、天使投资团队、冒险者公会）提供智力与组织根基；空间层（松山湖总部 + 高校联络点）提供物理载体与场景触达；资本层（天使投资资金池、双轨孵化机制）提供价值引擎。

### **1.3** 三条核心价值观

**新手村次元**以三条核心价值观构筑平台文化基石——**「快乐」**为引擎，**「互助」**为纽带，**「传承」**为使命，构成从个体进入到代际延续的完整价值体系。

#### **第一条：快乐优先**

**首先是快乐，其次都是其次。  
传统的成长体系将快乐定位为成功之后的奖励。**新手村次元将这一逻辑重构：被迫的坚持走不远，**发自内心的热爱才能穿越漫长的探索期。**快乐在此不是放纵，而是对内在驱动力的尊重——每位参与者首先需要**找到真正让自己愿意持续投入的事。**

#### **第二条：互助共创**

**我为人人，人人为我。  
传统的成长叙事强调个体奋斗。**新手村次元将这一逻辑重构：**成长不是一个人的长跑，而是一群人的接力。**当个体的努力被嵌入团队协作的网络，帮人解答、组织活动、分享经验便同时具备了双重属性——**于他人是支持，于自己是积累。**互助在此不是施舍，而是双向增值。在组织中被需要、被记录、被回馈，坚持便从孤立的自律转化为关系的自然牵引。

#### **第三条：薪火传承**

**前人栽树，后人乘凉。后人亦栽树，树多自成林。**

**传统的创业孵化追逐短期回报。**新手村次元将这一模式重构：**成长不是一代人的冲刺，而是代际间的接力。**传承在此不是复刻——不是要求后来者走与前辈相同的路，而是让前辈踩过的坑成为后来者的路标，**让验证过的经验成为探索的起点。**

### **1.4 文化落地的制度保障**

三条核心价值观并非停留在宣言层面，而是通过三层制度设计嵌入平台日常运营：

|          |                                      |
|----------|--------------------------------------|
| 价值观   | 对应制度                             |
| 快乐优先 | 任务化入口 + 成就系统 + 容错存档机制 |
| 互助共创 | 冒险币体系 + 身份体系 + 四大公会体系 |
| 薪火传承 | 长老导师制 + 大讲堂 + 经验资产化沉淀 |

三条价值观构成完整闭环：快乐驱动进入，互助驱动留存，传承驱动延续——**让青年愿意来、愿意留、愿意传。**

## 二、新手村视觉暗号

### **2.1** 村旗：Logo里的三层密码

**新手村次元Logo以像素化游戏风格为核心视觉语言，由三大要素构成：**

- **中文标准字「新手村次元」：**采用像素化造型处理，笔画结构模拟8-bit游戏字体风格。像素作为一种基础视觉单元，隐喻「每一个微小尝试都在构建更大的图景」——与平台价值观形成视觉呼应。

- **英文标识「NEWBIEVILLAGE DIMENSION」：**以等宽像素字体呈现，强化科技感与年轻属性。其中「NEWBIEVILLAGE」直译「新手村」，标识平台的成长起点定位；「DIMENSION」对应「次元」，指向超越物理空间的青年文化场域。

- **品牌标语「LEARN BY PLAYING, CREATE WHILE LEARNING」（玩中学，学中创）：**位于Logo组合下方，以简洁的无衬线字体呈现。这一标语以英文形式呈现，兼顾国际化表达与游戏化调性，精准概括了「玩→学→就→创」四维体系的核心逻辑。

Logo整体传递三层含义：像素风格标注「游戏式」的平台基因；「新手村+次元」的双词结构锚定品牌定位；中英双语组合兼顾本土识别与国际传播。

### 2.2 村子的颜色：亮色请你进来，深色让你留下

新手村次元的主KV配色方案围绕**「游戏化 + 青年文化 + 科技感」**三个维度构建：

|              |               |                                                              |
|--------------|---------------|--------------------------------------------------------------|
| **色彩角色** | **色系**      | **视觉语义**                                                 |
| 品牌主色     | 亮橙/活力黄   | 代表「快乐优先」价值观，传递热情、开放、低门槛的进入感       |
| 品牌辅色     | 深蓝/科技蓝   | 代表「专业」与「信任」，平衡活泼调性，标注平台的严肃创业属性 |
| 点缀色       | 荧光绿/像素绿 | 游戏UI经典配色，标注任务进度、成就解锁等交互元素             |
| 中性底色     | 深灰/炭黑     | 承载亮色主体内容，营造「像素屏幕」的沉浸视觉场域             |

配色策略的核心逻辑是**「亮色引入，深色沉淀」**——活力色调降低视觉进入门槛（对应快乐优先），深色底色承载专业内容，两者形成的视觉张力与平台「低门槛进入、高成长输出」的价值主张高度一致。

### **2.3** 视觉图腾：像素、头像与进度条

#### **2.3.1像素网格**

**像素网格是新手村次元最底层的视觉母题。**所有视觉元素——Logo、头像、图标、背景纹理——均以像素网格为基本构成单元。这一设计选择具有双重意涵：像素是数字世界的最小可见单位，隐喻「个体是平台的基础单元」；像素的排列组合可以构建任意复杂图形，隐喻「互助共创产生系统涌现」。

#### **2.3.2冒险者头像体系**

平台为每位核心成员设计像素风格头像——以统一的像素网格模板为基底，通过发型、服饰、配色等元素区分角色身份。头像体系与「旅行者→村民→执事→长老」四级身份挂钩：旅行者使用线框头像，村民获得填充色，执事解锁专属道具图标，长老拥有完整角色立绘。**头像不仅是身份标识，更是一种可感知的「成长可视化」工具。**

### **2.4** 冒险者角色矩阵：你会在谁身上看见自己

新手村次元的IP形象体系以**「冒险者角色矩阵」**为核心——每一位核心成员既是真实个体，也是IP角色。村长青舟澈、副村长少爷、吟游诗人小楼等核心成员均拥有预设的冒险者身份、角色定位和像素头像。这套体系区别于传统品牌IP的「单一吉祥物」模式，而采用「角色群像」策略：**冒险者的集合本身构成了平台IP的完整面容。**

IP体系的传播逻辑是**「去中心化的角色认同」**——不同背景的青年可以在不同角色中找到自我投射。想创业的人看见村长，喜欢技术的人看见少爷，热爱学术的人看见小楼，擅长社交的人看见小明。这种多角色IP策略扩大了品牌与目标受众的接触面积，同时以「群像」而非「偶像」的叙事方式，传递平台「互助共创」而非「个人英雄」的文化底层。

## FAQ

### Q1：新手村次元到底是什么？

**新手村次元**是立足东莞松山湖的**全国首创游戏式运营创业孵化平台**，由连续创业者黎圣传于2025年牵头创办。它不是一个传统意义上的创业园区或培训机构，而是一个将角色扮演游戏的底层设计逻辑——任务系统、成就体系、身份体系、公会协作——完整迁移到青年就业创业全流程的成长系统。

### Q2：和传统创业孵化器有什么不同？

**三个根本差异。**

1.  传统孵化器以项目筛选为起点，新手村次元**以人的成长需求为起点**；

2.  传统孵化器依赖导师单向输出，新手村次元**依赖互助共创网络**——帮人解答、组织活动、分享经验本身就是积累；

    第三，传统孵化器追求短期项目回报，新手村次元**以代际传承为使命**——每位走通路径的冒险者，都将经验沉淀为后来者的路标。

### Q5：三大价值观如何落地为制度？

每条价值观都有对应的制度硬支撑：

- **快乐优先**对应任务化入口、成就系统与容错存档机制；

- **互助共创**对应冒险币体系、四级身份体系（旅行者→村民→执事→长老）与四大公会体系；

- **薪火传承**对应长老导师制、大讲堂与经验资产化沉淀。

价值观不是口号，是可运转的机制。

### Q6：四大公会是做什么的？

**星火营地、启明书院、熔铁工坊、种光商会**，分别对应**「玩、学、就、创」**四阶段：星火营地是兴趣探索入口，启明书院负责知识学习与技能打磨，熔铁工坊承接真实商业任务与项目实战，种光商会对接创业孵化与资源整合。每个人根据自己的成长阶段进入对应的公会。
