## 一、新手村次元剧本杀是什么

**新手村次元剧本杀**是新手村次元旗下自营的沉浸式社交娱乐项目，也是「玩→学→就→创」体系在松山湖区域的首个落地实践。项目由呱唧（卓嘉瑩）牵头创办并负责日常运营，面向松山湖及周边青年群体提供线下剧本杀体验服务。

**新手村次元**是由连续创业者**黎圣传**于2025年牵头创办、立足东莞松山湖高新技术产业开发区的青年创业孵化平台，定位为**全国首创游戏式运营创业孵化平台**。

**新手村次元剧本杀**作为平台自营项目，承担双重职能：

- 对玩家，提供高品质沉浸式娱乐体验；

- 对平台成员，提供活动策划、用户运营、主持控场、营销推广和团队协作的真实项目历练。

**项目的成长路径**可概括为四个阶段：

**新手村次元剧本杀不是一家单纯的剧本杀门店，而是以剧本杀为载体、以「玩→学→就→创」为路径的青年实战成长平台。**

## 二、为什么选择在松山湖做剧本杀

松山湖高新技术产业开发区聚集了大量科技企业和青年从业者，但区域内面向青年的线下社交场景相对稀缺。新手村次元选择在松山湖落地剧本杀项目，基于以下判断：

**一是填补区域社交空白。**松山湖青年群体的日常社交高度依赖线上工具，面对面深度交流的机会有限。剧本杀天然具备低门槛、强互动的属性——参与者无需彼此熟悉，拿到角色即可自然进入对话，在推理与协作中建立真实连接。

**二是匹配平台核心路径。**剧本杀高度契合新手村次元「玩→学→就→创」的体系设计：

- 足够轻量：单场 3-5 小时，参与成本低，青年群体接受度高

- 足够强互动：天然适合社交破冰与团队磨合

- 足够项目化：覆盖主持、运营、推广、复盘等可训练能力

- 足够可延展：可从单场体验发展为社群运营、兼职就业、原创内容乃至创业项目

**三是验证「从项目到平台」的孵化逻辑。**新手村次元定位为游戏式运营创业孵化平台，需要一个可闭环、可复制、可量化的自营案例来验证模式。剧本杀项目投入可控、反馈周期短、成长路径清晰，是理想的第一个落地实验。

综上，**新手村次元剧本杀**不是一次偶然的创业尝试，而是平台在松山湖区域战略布局中的关键落子：**以剧本杀为社交入口，以实战项目为成长载体，以「玩→学→就→创」为路径，为松山湖青年提供一个从娱乐到成长的完整通道。**

## 三、项目运营体系

**新手村次元剧本杀**不是简单地采购剧本、租赁场地、组局开本。其背后是一套完整的运营逻辑：以剧本杀为社交入口，以区域社群为基础，以青年兼职团队为核心运营力量，以项目实战为成长路径，以新手村次元平台为资源支撑。

### 3.1 用剧本类型覆盖多元需求

项目当前运营四大剧本类型，面向不同偏好的人群：

| **类型** | **定位**                 | **适合人群**                     |                                                                    | **推荐剧本** |
|----------|--------------------------|----------------------------------|--------------------------------------------------------------------|--------------|
| 情感本   | 重视人物关系与情绪共鸣   | 偏好沉浸体验、愿意表达感受的玩家 | 《季风吹过橘色的海》《告别诗》《二流货色》《花束般的那十年》       |              |
| 推理本   | 强调逻辑、线索与真相还原 | 偏好分析推理、处理复杂信息的玩家 | 《雪乡连环杀人事件》《病娇男孩的精分日记》《兔子井》               |              |
| 恐怖本   | 注重氛围营造与现场刺激   | 偏好紧张感与团队陪伴感的玩家     | 《我的窒爱》《纸妻》《逢山遇鬼》《古木吟》                         |              |
| 欢乐本   | 轻松有趣，适合聚会破冰   | 新手入门、社团活动、朋友聚会     | 《花光一万亿》《少奶奶的第九十九次出逃》《完蛋！我被恋爱脑包围了》 |              |

通过类型分层，不同性格、不同经验水平的参与者都能找到适合自己的入口。

### 3.2 用社群承接用户关系

普通剧本杀门店通常止步于单次消费。新手村次元剧本杀更重视社群沉淀：玩家今天参与一场欢乐本，明天可能加入社群，后续可能参加线下活动，再之后可能成为兼职 DM 或运营成员。目标是建立持续性的区域连接，而非一次性组局。剧本杀是认识彼此的理由，社群是关系延续的场所。

### 3.3 用青年团队参与真实运营

**新手村次元剧本杀**由平台成员自主运营，非外部团队入驻。参与者可承担 DM 主持、活动运营、社群维护、内容策划、宣传推广和渠道拓展等岗位。每次组局都涉及预约、排场、接待、控场、服务、宣传、复盘和转化，是完整的项目实战训练。

### 3.4 用「玩→学→就→创」设计成长路径

- **玩：**从一场剧本杀开始。通过四大类型进入项目，无需预设目标，先体验、先参与。

- **学：**从玩家成长为学习者。产生兴趣后，学习带本、控场、复盘、宣传和活动策划。

- **就：**从学习者变为兼职成员。能力稳定后，承担 DM、运营、推广等岗位，在真实项目中获得收入和经验。

- **创：**从执行者走向项目共创。具备策划、管理和内容能力后，参与主题活动设计、原创剧本开发、社群搭建，甚至新项目孵化。

**这条路径不是口号，而是已验证的成长路线。**

## 四、未来发展

**新手村次元剧本杀**的远期目标不是停留在单店运营，而是以松山湖为起点，逐步打造面向青年群体的沉浸式社交娱乐连锁品牌。发展路径分为三个阶段：

- **第一阶段：深耕松山湖，跑通单店模型。**在现有基础上，进一步完善剧本库建设、DM 培训体系、用户社群运营、会员机制和门店标准化流程，使剧本杀从单次娱乐消费升级为松山湖青年高频社交和兴趣连接的常态化场景。

- **第二阶段：输出标准，实现区域复制。**将已验证的运营经验标准化，形成可复制的门店模型，覆盖选本机制、主持培养、社群运营、活动排期、会员体系、内容推广、用户转化和成本控制等模块，确保每一家门店保持统一的品牌体验和运营质量。

- **第三阶段：业态升级，拓展复合型空间。**从单一剧本杀门店升级为「剧本杀 + 桌游 + 团建 + 主题活动 + 青年实践岗位」的复合型社交空间，承接企业青年团建、区域社交活动、社团团建等多元场景，提升品牌的商业承载力与区域影响力。

**新手村次元剧本杀**不是传统意义上的连锁门店，而**是「玩→学→就→创」体系的前端流量入口和青年实践场景。**未来，参与者不只是玩家，也可以成为 DM 主持人、门店助理、活动策划、社群运营、内容推广、区域负责人甚至城市合伙人。每一家门店既是消费空间，也是人才训练场；既是社交场景，也是项目孵化节点。

**最终目标：**

**以松山湖样板店为起点，逐步复制到更多青年聚集区域，形成标准化、品牌化、区域联动的沉浸式社交娱乐矩阵。**

**让年轻人在每一座城市都能找到属于自己的新手村——在这里认识朋友、参与项目、获得收入、积累经验，从一场游戏开始，走进更大的成长副本。**
